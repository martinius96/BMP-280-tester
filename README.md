# Adafruit BMP280 tester (Barometric Pressure, Temperature Sensor) #
## Zapojenie - Wiring

ESP32 Devkit-c V4|BMP280
:----------: | :----------:
3.3V|Vcc
GND|GND
GPIO21 (Hardware SDA)|SDA
GPIO22 (Hardware SCL)|SCL

* Knižnica používa adresu (0x76) pre čínske moduly BMP280
